# Morning Dew Meal Planner

A comprehensive meal planning application built with modern web technologies.

## Getting Started

### Prerequisites

Make sure you have Node.js & npm installed - [install with nvm](https://github.com/nvm-sh/nvm#installing-and-updating)

### Installation

```sh
# Clone the repository
git clone https://github.com/The-Blue-Bale/Meal-Planner-Pro.git

# Navigate to the project directory
cd Meal-Planner-Pro

# Install dependencies
npm i

# Start the development server
npm run dev
```

## Technologies Used

This project is built with:

- Vite
- TypeScript
- React
- shadcn-ui
- Tailwind CSS

## Features

- Weekly meal planning
- Recipe management
- Person-specific meal assignments
- Calendar view for meal planning
- Recipe builder and customization
- Meal sharing capabilities

## Development

### Available Scripts

- `npm run dev` - Start development server
- `npm run build` - Build for production
- `npm run preview` - Preview production build
- `npm run lint` - Run ESLint

## Deployment

Build the project using `npm run build` and deploy the `dist` folder to your preferred hosting platform.
